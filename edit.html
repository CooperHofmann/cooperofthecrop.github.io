How to Add/Remove/Reorder Images

Step-by-Step Instructions:

  1. Add your image files to the appropriate folder in /images/
  2. Open /js/config.js in a text editor
  3. Add the filename to the images array for the category
  4. Images appear in the order they're listed in the array
  5. Save the file and refresh the portfolio page to see changes
  6. Commit and push changes to GitHub to deploy

Deploy to GitHub Pages

After making changes:

  1. Save all modified files
  2. Open terminal/command prompt in your project folder
  3. Run: git add .
  4. Run: git commit -m "Update portfolio images"
  5. Run: git push
  6. Wait 1-2 minutes for GitHub Pages to deploy
  7. Visit your live site to see the changes
